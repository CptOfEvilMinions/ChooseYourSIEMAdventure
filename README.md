# Choose your own SIEM adventure
Repo of configs for the three major SIEMs. 

## Snakeoil certs
1. `openssl req -x509 -nodes -days 3650 -newkey rsa:2048 -keyout conf/tls/docker.key -out conf/tls/docker.crt`

## Supported versions
* `Ansible v2.11+`
* `Ubuntu 20.04 64-bit`
